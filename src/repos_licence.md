**[This](https://github.com/clearmatics/github_license_search) repo contains the python script and instructions to use it.**

Searches github for license (or another file) on all branches. Returns list of repos that dont have a license and a list of repos with one line licences, e.g. `Copyright (C) 2019 Clearmatics Technologies Ltd - All Rights Reserved`, and a list of repos that have a license file in an unusual branch. The script takes a few minutes to run so the results are displayed below.
